Import & Export Data
Sambungkan ke Google Sheets, ekspor ke Excel, atau impor data
☁️ SAMBUNGKAN KE GOOGLE SHEETS Belum tersambung

📋 Cara Setup (sekali saja, ~10 menit):

  1. Buka sheets.google.com → buat spreadsheet baru, beri nama Buku Kas TKI
  2. Klik menu Extensions → Apps Script
  3. Hapus semua kode yang ada, paste isi file BukuKas_Backend.gs yang sudah lo download
  4. Klik ikon Save 💾 → beri nama project BukuKas Backend
  5. Klik Deploy → New Deployment → Type: Web App
  6. Execute as: Me · Who has access: Anyone → klik Deploy
  7. Copy URL deployment yang muncul → paste di kotak di bawah ini
